Multiplayer fighting game.

Clone repo.

Navigate to directory.

    sudo npm install
    sudo npm install -g bower
    bower install
    node server.js

Navigate to http://localhost:3000/ to play

Navigate to http://localhost:3000/ai for a programmable bot
